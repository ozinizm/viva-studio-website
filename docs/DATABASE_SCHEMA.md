- users
- roles
- permissions
- site_settings
- hero_sections
- services
- service_categories
- gallery_items
- blog_posts
- blog_categories
- campaigns
- reservation_requests
- contact_messages
- faqs
- testimonials
- seo_settings
- legal_pages
- media_assets
