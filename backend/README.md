# Backend Plan

Backend bu ilk aşamada aktif değildir. Geliştirme tamamen frontend (mock servislerle) üzerinde yapılmıştır.

İleride kurulması planlanan altyapı:
- **Framework:** FastAPI veya NestJS
- **Veritabanı:** PostgreSQL
- **Yetkilendirme:** JWT tabanlı Auth ve Role-Based Access Control (RBAC)
- **API Modülleri:** 
  - Admin CRUD işlemleri (Hizmetler, Galeri, Blog vb.)
  - Rezervasyon ve İletişim Formu API'leri
  - SEO ve site ayarları yönetimi
